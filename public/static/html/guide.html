1 Map

XAP displays two levels of maps, which are campus map and floor map in a building. Campus map is displayed in the home page, and floor map can be viewed by clicking indoor button in the information panel of the building.

1-1 Location

XAP categorizes locations (or items) in campus into three different types, which are building, room and facility. Buildings are only provided in the campus map, whereas rooms are only displayed in floor maps. Both levels of map have facility information.

Every item in a map should have been marked in a round icon. However, since all the buildings in the campus map and all the rooms in most floor maps in the north campus are provided, icons of building or room in these maps will not be displayed explicitly.

Try to click or tap on the icon or the area of an item to get its information.

1-2 Zoom

For the desktop version, maps are zoomed by mouse. There are three ways to scale the map: zoom button, mouse wheel and double click on the map.

For the mobile version, maps are zoomed by the gesture of fingers on the screen. Maps can be zoomed according to the movements of two fingers or double tap of one finger to zoom in.

Notice: In most browsers of mobile devices, the zoom gesture of fingers may result in the scale of whole web page instead of map component only. Make sure the web page is displayed in its initial scale while zooming the map.

3 Search

Search panel is placed on the left side of web page in desktop version, and at the bottom in the broser of mobile devices. Type words in the input bar to get results.

Currently search engine only enables English and Chinese search. Each language environment only enables its own language search. Spanish environment uses english search as default.

The search engine categorizes results and displays the top three items in each category. User is able to click on "View More Results" to get other results in a category.

4 Other Components

XAP provides additional functions in the system to meet specific demand. Campus and floor maps include different components.

4-1 Menu

Menu contains serveral services which can be manipulated in every map.

4-1-1 Language Setting

Currently XAP provides three language environments (EN as English, ZH as simplified Chinese, and ES as Spanish) for different target users. English is the primary language in the system. User is able to change languages in this component.

4-1-2 Help

View user guide to learn basic operation of XAP.

4-1-3 Virtual Button

Click on this button to display a virtual button and hides all the components except search panel on the map. Click on the virtual button to quit this mode.

4-2 Zoom Button (Desktop Version Only)

Click on zoom in or zoom out button to scale the map.

4-3 Location Button (Campus Map Only)

User is able to trigger real-time location in the campus map. Orientation of device is provided in some browsers of mobile devices.

4-4 Floor List (Floor Map Only)

This component indicates the building where current floor map locates with abbreviation of the building. User can click on current floor number to get the floor list of the building and then go to another floor.

4-5 Campus Button (Floor Map Only)

Go to campus map.

4-6 Occupation Button (Floor Map Only)

This function enables user to get room occupation to check which rooms are occupied by courses according to timetables in current floor at a specific time. It provides time selector for user to check room occupation in every possible moment in current semester.

Group icon will be displayed at the center of the room area indicating the room is occupied at the moment.

Notice: When occupation mode is activated, it is not allowed to select locations on the map. User is able to click the button or choose locations from search panel to quit the mode.

4-7 Entrance Button (Floor Map Only)

This function enables user to display building access on the map. Available entrances will be highlighted on the map with arrows. Arrows of opening gates are displayed with bouncing animation.

According to the different closing time for entrances, there are several types of entrance arrow:
  • Open 24 hours
  • Open from 7:00 to 17:30
  • Open from 7:00 to 11:00
  • Entrance not available
